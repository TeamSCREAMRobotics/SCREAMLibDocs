# SCREAMLib

SCREAMLib is Team SCREAM Robotics' shared Java toolkit for FRC robots. It wraps recurring work around CTRE Phoenix 6 devices, PID configuration, geometry, alliance-aware field logic, vision, simulation, logging, dashboard mechanisms, paths, and unit-safe lengths.
